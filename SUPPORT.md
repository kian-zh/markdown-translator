# Support / 支持

## Bug reports and feature requests / 缺陷与功能建议

Please open an issue at [GitHub Issues](https://github.com/kian-zh/markdown-translator/issues).

请通过 [GitHub Issues](https://github.com/kian-zh/markdown-translator/issues) 提交问题或功能建议。

For a translation failure, include the extension version, VS Code or Cursor version, operating system, and the exact error message. Do not include confidential document content.

遇到翻译失败时，请提供扩展版本、VS Code 或 Cursor 版本、操作系统和完整错误信息；请勿提交机密文档内容。
